Mobile Details

The Nokia Lumia 720 is another Windows based smartphone from Nokia. The mid-level device comes power packed with some exciting features. Given below are the features and specifications of the device.

Features and Specifications:

The Lumia 720 runs on Microsoft Windows Phone 8 operating system which can be upgraded to Windows Phone 8 Black OS. The device is powered by 1 GHz dual-core Qualcomm processor with Adreno 305 GPU. In terms of memory, the Nokia Lumia 720, is similar to Lumia 625 as the smartphone hosts 512 MB RAM with an internal storage capacity of 8 GB which can be further expanded up to 64 GB using a microSD card.

Screen and Design:

The Lumia 720 has a 4.3-inch IPS LCD capacitive touch screen with qHD (800 x 480 pixels) screen resolution which produces an effective pixel density of 217 PPI. The device also features Corning’s Gorilla Glass 2 protection and also a Clear Black display which helps you to get more clarity in bright sunlight. The smartphone measures 127.9 x 67.5 x 9 mm and weighs about 128 grams. The smartphones comes in five colour variants namely Yellow, Cyan, Black, White and Red.

Camera and Multimedia:

The device has an in-built 6.1 MP primary rear camera with LED flash using which one can capture quality images and high definition videos. The device is also equipped with a 1.3 MP front facing camera which one can use to click self-portraits and videos. Multimedia options that come along with the Windows based smartphone are MP3, MP4, WMV, WAV, WMA etc.

Software and Features:

The Nokia Lumia 720 runs on Microsoft Windows Phone 8 operating system which can be upgraded to Microsoft Windows Phone 8 Black operating system. The smartphone also features pre-loaded regular applications like image viewer & editor, document viewer, Predictive text input, 7 GB free Skydrive storage, Here Maps, Bing search etc.

Battery and Connectivity:

The device is powered by a non-removable Li-Ion 2000 mAh battery which supports a talk time of about 23h and music playback up to 79h. In terms of connectivity, one can always afford to stay connected as the device includes 3G, GPRS, EDGE, Wi-Fi 802.11 b/g/n, Wi-Fi hotspot, Bluetooth 3.0, NFC and USB 2.0 port.
